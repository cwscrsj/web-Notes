## 作用域

### 1. 作用域分类

#### 1. 全局作用域

一个 html 页面就是一个全局作用域，打开页面的时候, 作用域就生成了, 直到关闭页面为止

#### 2. 局部作用域

只有函数生成私有作用域

 每一个函数就是一个局部作用域

### 2. 作用域链

+ 只要有代码就是至少一个作用域

+ 写在函数内部的局部作用域

+ 若函数内还有函数，则在作用域内可以再诞生一个作用域

+ 根据 <font color='red'>内部函数可以访问外部函数变量</font>，用链式查找决定哪些数据能被内部函数访问，称为作用域链

  <font color='red'>	**就近原则**</font>

### 3. 案例

```javascript
 function f1() {
	var num=123;
     function f2() {
         console.log(num);
	}
     f2();
}

var num=456;
f1();

/* 结果是 */
num=123;
```
